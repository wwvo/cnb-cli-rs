# cnb mission delete

```
cnb mission delete <mission> [options]
```

删除任务集。

## 参数

| 参数        | 缩写 | 说明             |
|-------------|------|------------------|
| `<mission>` |      | 任务集路径       |
| `--yes`     | `-y` | 跳过确认提示     |

## 示例

```bash
cnb mission delete my-org/sprint-2025-q1
cnb mission delete my-org/sprint-2025-q1 --yes
```

# cnb mission view get

```
cnb mission view get <mission> --id <view_id>
```

获取视图配置详情。

## 参数

| 参数             | 说明         |
|------------------|--------------|
| `<mission>`      | 任务集路径   |
| `--id <view_id>` | 视图 ID      |

## 示例

```bash
cnb mission view get my-org/sprint-2025-q1 --id view-001
cnb mission view get my-org/sprint-2025-q1 --id view-001 --json
```

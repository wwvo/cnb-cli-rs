# cnb issue

Issue 管理，查看、创建和管理仓库的 Issue。

## 可用命令

- [cnb issue list](/issue/list) — 列出仓库的 Issue
- [cnb issue mine](/issue/mine) — 列出与我相关的 Issue
- [cnb issue create](/issue/create) — 创建 Issue
- [cnb issue close](/issue/close) — 关闭 Issue
- [cnb issue comment](/issue/comment) — 创建 Issue 评论
- [cnb issue exist](/issue/exist) — 检查 Issue 是否存在
- [cnb issue download](/issue/download) — 下载 Issue 为 Markdown 文件
- [cnb issue assigners](/issue/assigners) — Issue 处理人管理

## 示例

```bash
$ cnb issue list
$ cnb issue create
$ cnb issue mine
```

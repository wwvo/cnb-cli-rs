# cnb member repo-remove

```
cnb member repo-remove <username> [options]
```

移除仓库成员。

## 参数

| 参数         | 缩写 | 说明             |
|--------------|------|------------------|
| `<username>` |      | 用户名           |
| `--yes`      | `-y` | 跳过确认提示     |

## 示例

```bash
cnb member repo-remove zhangsan
cnb member repo-remove zhangsan --yes
```

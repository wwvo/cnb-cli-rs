# cnb member repo-list

```
cnb member repo-list [options]
```

列出仓库直接成员。

## 参数

| 参数             | 缩写 | 说明         |
|------------------|------|--------------|
| `--role <role>`  | `-r` | 按角色过滤   |
| `--search <kw>`  | `-s` | 搜索成员     |

## 示例

```bash
cnb member repo-list
cnb member repo-list --role Developer
cnb member repo-list --search zhang
cnb member repo-list --json
```

# cnb member repo-inherited

```
cnb member repo-inherited [options]
```

列出仓库继承成员。

## 参数

| 参数             | 缩写 | 说明         |
|------------------|------|--------------|
| `--role <role>`  | `-r` | 按角色过滤   |
| `--search <kw>`  | `-s` | 搜索成员     |

## 示例

```bash
cnb member repo-inherited
cnb member repo-inherited --role Master
cnb member repo-inherited --json
```

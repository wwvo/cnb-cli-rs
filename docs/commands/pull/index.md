# cnb pull

Pull Request 管理，查看、创建、更新和合并 PR。

## 可用命令

- [cnb pull list](/pull/list) — 列出与我相关的 Pull Request
- [cnb pull create](/pull/create) — 创建 Pull Request
- [cnb pull update](/pull/update) — 更新 Pull Request
- [cnb pull merge](/pull/merge) — 合并 Pull Request

## 示例

```bash
$ cnb pull list
$ cnb pull create
$ cnb pull merge
```

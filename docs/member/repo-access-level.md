# cnb member repo-access-level

```
cnb member repo-access-level [options]
```

查看自己在仓库的权限。

## 参数

| 参数                | 说明             |
|---------------------|------------------|
| `--include-inherit` | 包含继承权限     |

## 示例

```bash
cnb member repo-access-level
cnb member repo-access-level --include-inherit
cnb member repo-access-level --json
```

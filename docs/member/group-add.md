# cnb member group-add

```
cnb member group-add <username> --group <group> --role <role>
```

添加组织成员。

## 参数

| 参数              | 缩写 | 说明         |
|-------------------|------|--------------|
| `<username>`      |      | 用户名       |
| `--group <group>` | `-g` | 组织路径     |
| `--role <role>`   | `-r` | 权限等级     |

## 示例

```bash
cnb member group-add zhangsan --group myorg --role Developer
```

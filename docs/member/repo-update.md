# cnb member repo-update

```
cnb member repo-update <username> --role <role>
```

更新仓库成员权限。

## 参数

| 参数             | 缩写 | 说明               |
|------------------|------|--------------------|
| `<username>`     |      | 用户名             |
| `--role <role>`  | `-r` | 新权限等级         |

## 示例

```bash
cnb member repo-update zhangsan --role Master
```
